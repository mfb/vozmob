This module provides a simple interface to SMS messaging via e-mail gateways.

It supports both sending and receiving of email messages, and includes a simple
test module for incoming messages.

The test module includes examples of some of the hooks available in the module.
See the code notes in the test module for usage details.

See INSTALL.txt for installation instructions.
